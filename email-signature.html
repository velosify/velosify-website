How to install in Gmail

  1. Pick the version you want below.
  2. Select the entire signature block (click just before "Nolan Johnson" and drag to the end of the phone number).
  3. Copy it (Cmd+C / Ctrl+C).
  4. In Gmail: gear icon → See all settings → General tab → scroll to Signature.
  5. Click Create new, name it "Velosify", then paste into the signature editor box (Cmd+V / Ctrl+V).
  6. Set defaults below the editor: "For New Emails" and "On Reply/Forward" both to Velosify.
  7. Scroll to the bottom and click Save Changes.
